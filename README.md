## Getting Started
### Prerequisites
- [Git](https://git-scm.com/)
- [Node.js and npm](https://nodejs.org/) Node ^14.15.4, npm ^6.14.10
### Developing
1. Run `npm install` to install server dependencies.
2. Run `node server` to render server on localhost:3030
